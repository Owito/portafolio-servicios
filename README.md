# Portafolio de Servicios

Landing onepage construida con Next.js 16 y Material 3 (MUI), enfocada en presentar servicios digitales con una experiencia visual moderna y optimizada para conversion.

## Enlaces

- Produccion: https://portafolio-servicios-rho.vercel.app
- Repositorio: https://github.com/Owito/portafolio-servicios

## Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Material UI (Material 3)
- Tailwind CSS v4 (soporte base)

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Desarrollo local

1. Instala dependencias:

   ```bash
   npm install
   ```

2. Inicia el entorno de desarrollo:

   ```bash
   npm run dev
   ```

3. Abre http://localhost:3000
